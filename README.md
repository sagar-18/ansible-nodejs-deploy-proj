# ansible-nodejs-deploy-proj
Setting up Nodejs Application using ansible

OS: Ubuntu 22 (Control Node)

Step 1: Update OS

sudo apt update 

Step 2: Install Ansible

sudo apt install ansible

Step 3: Check if it is installed

sudo ansible --version
